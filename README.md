# aisddot2000

repository-agnostic instructions for specification driven development assisted by ai agents.

start here: [full phase-by-phase guide v1](docs/v1/README.md)

## pipeline

```txt
phase 01: idea -> feature description
phase 02: feature description -> outline
phase 03: outline -> specs
phase 04: specs -> review
phase 05: review -> implementation plan
phase 06: plan -> implementation
```

each phase has its own instructions, templates or checklists, and copy-paste prompts.
